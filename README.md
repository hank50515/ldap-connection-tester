# LDAP Connection Tester

Multi-platform LDAP/Active Directory connection testing tool.

## Supported Platforms
- ✅ Windows (PowerShell)
- ✅ Linux/macOS (Bash)

## Quick Start

### Windows
```powershell
.\powershell\Test-LDAPConnection.ps1
```

### Linux/macOS
```bash
./bash/test-ldap-connection.sh
```
